Introduction

Workflow CLI is a toolkit designed for kick-starting React web projects on the Availity Portal.#

Similar to CRA

Get Started#

There are 2 main ways to get started with Workflow CLI:

  1. Tutorial: Step-by-Step instructions on how to install and start a project. This also includes details on the project structure as well as what each file/folder is used for.
  2. Quick Start: The TLDR guide to kickstarting a project from start to finish.
